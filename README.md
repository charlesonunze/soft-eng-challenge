## Run Locally

Clone the project

```bash
  git clone https://github.com/charlesonunze/soft-eng-challenge
```

Go to the project directory

```bash
  cd soft-eng-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  make dev
```

Watch for file changes

```bash
  make watch
```

Run tests

```bash
  make test
```
